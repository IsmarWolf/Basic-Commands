Useful commands CSS

Universal Selector (*) - Parent of all

margin: 0: - Removes all margin spacing around elements

box-sizing: border-box - Includes padding and border in element's total width/height

font-family: Arial, sans-serif - Sets Arial as primary font, fallback to system sans-serif

Layout Properties

display: flex (2,3) (header, nav) - Creates flexible box layout

justify-content: center (2) (nav) - Centers items horizontally in flex container

gap (2) (nav) - Creates spacing between flex items

flex-wrap: wrap (3) (pdts) - Allows items to wrap to next line

flex-direction: column (3) (pdts) - Changes flex direction to vertical

align-items: center (3) (pdts) - Centers items vertically in flex container

Spacing Properties

padding (2,3) (header, main) - Creates inner spacing within elements

margin-top (2,3) (nav, footer) - Creates spacing above elements

margin-bottom (2,3) (main h1, pdt h2) - Creates spacing below elements

width (2,3) (pdt, logo) - Sets element width

height (2,3) (pdt img, logo) - Sets element height

max-width (2,3) (pdt) - Sets maximum width limit

Visual Properties

background-color (2) (body, header) - Sets background color

color (2,3) (header, pdt h2) - Sets text color

border-radius (2) (pdt, pdt img) - Rounds corners of elements

box-shadow (2) (pdt) - Adds shadow effect to elements
